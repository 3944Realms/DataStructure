# 这是我的线性结构数据作业代码仓库

* 作业源码位于根目录下以`Exp<单元实验数>_<实验项>.cpp`命名

* 库源码由标准库和自写数据结构库组成，其中自写数据结构库位于`basicDataStructure`目录下
    > * SqList - 顺序表
    > * LinkList - 单链表
    > * DLinkList - 双链表
    > * CLinkList - 循环单链表
    > * CDLinkList - 循环双链表
    > * SqStack - 顺序栈
    > * LinkStack - 链栈
    > * SqQueue - 顺序队列
    > * LinkQueue - 链队列
    > * BTTree - 二叉树

构建工具为Cmake
